Getting Started

Clone demi repo

git clone https://github.com/enytc/demi.git

Enter in demi directory

cd demi

Install dependencies

npm install

Configure the settings in api/config

Run demi

npm start

Test your demi app

npm test

Using Yeoman Generator

Install Demi.js generator

npm install -g generator-demi

Run Demi.js

yo demi
